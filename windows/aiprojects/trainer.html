AI/ML Projects 2/5 - Pokemon Trainer

GitHub Link

Myself and a partner were tasked with designing an AI agent that could optimally play and win Pokemon battles using the Stochastic Adversarial Search algorithm.

To accomplish this, we designed a tree data structure made up of chance nodes and deterministic nodes to predict game outcomes based on our moves and the opponents moves.

We implemented a utility heuristic and a-b pruning to guide our agent to make optimal moves and make the decision-making process more efficient.

Furthermore, we discovered inefficiences in the given game engine, and engineered solutions to bypass them to create an efficient agent that can see up to 7 moves ahead in-game to make better informed tactical decisions.

Tech Used:

-Stochastic Adversarial Search algorithm

-Alpha-Beta Pruning algorithm

-Utility heuristic curated for Pokemon battling

-Pokemon Battle Engine designed by Professor Andrew Wood at BU

-Github for version control and team collaboration

-Visual Studio Code

To satisfy my own curiosity I wanted to see if it was possible to design a Pacman AI agent that could outperform a human---all without the use of power pellets.

Without using any game engine, I built a Pacman game in VS Code, then implemented authentic ghost AI to try and replicate the original Pacman.

From there, I implemented the A star algorithm and curated a utility hueristic to allow Pacman to navigate the maze, and continuously beat levels without a game over.

Tech Used:

-A star algorithm

-Utility heuristic curated for Pacman's survival

-Greedy algorithms for decision making

-Java Data Structures & JFrame & JPanel class

-Github for version control

-Visual Studio Code
